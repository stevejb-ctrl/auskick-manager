THE PURE FUNCTION (algorithm to implement exactly):
```
deriveSubIntervalSeconds(periodSeconds, floorSeconds):
  // Guard: can't sub more often than the floor; a period at/under the floor
  // gets a single window.
  if periodSeconds <= floorSeconds: return periodSeconds
  // 1. Smallest PROPER clean divisor in [floor, periodSeconds) — most-frequent
  //    fair rotation that lands subs on clean fractions of the period.
  for D = ceil(floorSeconds) .. periodSeconds - 1:
      if periodSeconds % D === 0: return D
  // 2. Near-even fallback: no proper divisor >= floor (the only clean divisor
  //    >= floor is periodSeconds itself). Split into the most near-equal
  //    windows that each stay >= floor.
  N = floor(periodSeconds / floorSeconds)      // max windows with each >= floor
  if N >= 2: return Math.round(periodSeconds / N)
  return periodSeconds
```
All inputs are integer seconds (sport periods are multiples of 60); return an
integer. Keep the function PURE (no imports, no config reads) so it is trivially
unit-testable and reusable by the projector/configs.
