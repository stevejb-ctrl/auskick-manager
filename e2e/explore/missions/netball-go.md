You're coaching netball "Go" — kids aged 8 to 10. Seven on
court, full position set (GS, GA, WA, C, WD, GD, GK). Real
match format with quarters. This is the age where kids start
identifying with positions.

What's distinctive at Go:

- First age with the full 7-position model. Coaches teach kids
  what each position means; the app's labels need to back that
  up (use full names like "Goal Shooter" not just "GS").
- Court restrictions become teaching moments. A GS who runs
  into the centre third is a learning opportunity, not a
  failure. The app's enforcement should support this — error
  messages need to be readable.
- Some kids are shooters, some defenders, most still finding
  out. Rotation needs to cycle them through positions to
  give each one a try.
- Centre passes and turnovers happen rapidly; the score can
  flip 0-2 in a couple of minutes. Live scoring needs to keep
  up.
